# Exercise 20: Number Statistics Calculator

## Instructions
In this exercise, you will:
1. Read 10 numbers from the user (using `prompt()`)
2. Calculate and display:
   - The maximum number
   - The minimum number
   - The average of all numbers

## Example
If the user enters: 15, 7, 23, 5, 9, 14, 3, 8, 12, 10

Expected output:
```
Maximum number: 23
Minimum number: 3
Average: 10.6
```

## Tips
- Remember to convert user inputs to numbers
- Think about how to keep track of the largest and smallest numbers
- Consider how to calculate the average of a set of numbers
